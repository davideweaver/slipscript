cgihtml

version 1.51

a set of CGI and HTML routines written in C

Recently named Top 2 CGI Programming Tools by PC Computing (August 1996).

by Eugene Eric Kim, <eekim@hcs.harvard.edu>

This documentation is also available on http://hcs.harvard.edu/~eekim/web/cgihtml/cgihtml.html

Documentation last updated: August 10, 1996

Where to Find cgihtml

You can download cgihtml from ftp://hcs.harvard.edu/pub/web/tools/cgihtml.tar.gz. cgihtml is also available in UNIX compressed and pkzipped formats.

Useful information about cgihtml is available at the cgihtml home page, located at http://hcs.harvard.edu/~eekim/web/cgihtml/.
