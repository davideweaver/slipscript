Installing SlipScript on Microsoft IIS

For this example installation, we are assuming the Website webserver is installed in c:\website.
  1. Download SlipScript to a temporary directory.
  2. Unzip the file you just downloaded.
  3. Create a directory to hold the IS_SLIP.DLL (for this example we will make it c:\inetsrv\sscripts). Also, the directory must have Execute permissions
  4. Move IS_SLIP.DLL to the directory you just created.
  5. In the registry under w3svc\parameters\Script Map create the following value: .slip=c:\inetsrv\is_slip.dll
  6. Stop and Restart the web server in the IIS Manager.
  7. The scripts you run must be in a virtual directory with Execute permission.
