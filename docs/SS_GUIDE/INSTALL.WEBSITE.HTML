Installing SlipScript on O'Reilly's Website server

For this example installation, we are assuming the Website webserver is installed in c:\website.
  1. Download SlipScript to a temporary directory.
  2. Unzip the file you just downloaded.
  3. Run CONVERT.BAT to change to long filenames.
  4. Create a directory to hold your SlipScript scripts (for this example we will make it c:\website\sscripts).
  5. Move SLIPSCRIPT.EXE, TEST.SLIP and STRING.SLIP to the directory you just created.
  6. In File Manager, create an association for the extension .SLIP with c:\website\sscripts\slipscript.exe.
  7. Goto the Website Server Properties page, on the Mapping tab choose the Standard CGI radiobutton. Now enter the information to map to the new directory you created in step 4.
  8. Now goto the CGI tab, make sure the Standard CGI Shell box has an asterisk in it and the Exec Template is ~s ~p.
  9. Stop and Restart the web server in Control Panel.
